## Install

### npm

```
$ npm install @polyfiller/test262
```

### Yarn

```
$ yarn add @polyfiller/test262
```

### pnpm

```
$ pnpm add @polyfiller/test262
```

### Run once with npx

```
$ npx -p @polyfiller/test262 test262
```
